## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/one-page-binder.git
   cd one-page-binder
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:3000`

### Building for Production

```bash
# Build the application
pnpm build

# Start production server
pnpm start
```

### Tauri Desktop App

```bash
# Install Tauri CLI
pnpm add -g @tauri-apps/cli

# Run Tauri development
pnpm tauri dev

# Build Tauri app
pnpm tauri build
```
